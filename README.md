# Database--From_Scratsh_To_PHP
Database Design &amp; Implementation

EFORIA is a relational database system developed in MySQL to manage and organize tax-related information for citizens, including declarations, taxes, properties, bank accounts, and audits. The system simulates the core functions of a tax authority and enforces data integrity, business rules, and historical tracking.

This project was developed as part of a university database course, demonstrating skills in database design, SQL, triggers, stored procedures, and relational modeling.
